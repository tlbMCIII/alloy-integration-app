# About 
This project is a lightweight full-stack application that integrates with the Alloy API to manage customer onboarding evaluations. It provides a simple front-end interface for submitting application data and a secure backend for handling API communication, evaluation, and review responses.

#Overview
The Alloy Integration App allows users to:

> Submit onboarding applications with applicant and address details

> Review or approve applications based on Alloy API evaluations

> View real-time responses from the Alloy API in a clean, user-friendly UI

>The goal of this app is to demonstrate integration between a React frontend and an Express.js backend communicating with an external API (Alloy).

# Tech Stack 

Frontend:

>React (with functional components and hooks)

>Axios for API requests

>Tailwind CSS for styling

Backend:

>Node.js with Express.js

>Axios for HTTP requests to Alloy API

>CORS and dotenv for configuration and environment management

Hosting:

>Render: Backend (Express server)

>Render: Static site hosting for frontend build
